## O que são LinkedIn Skill Assessments?

4

Os **Skill Assessments** são testes técnicos dentro do **LinkedIn** que permitem validar suas habilidades em diversas áreas.

Após passar no teste, você recebe um **badge (selo de habilidade)** exibido no seu perfil.

Isso mostra para recrutadores e empresas que você **não apenas diz que sabe**, mas **provou que sabe**.

------

# Como funciona o Skill Assessment

O processo é simples e direto.

### 1️⃣ Escolha uma habilidade

Dentro do LinkedIn existem avaliações para diversas áreas:

- programação
- cloud
- data analytics
- DevOps
- bancos de dados
- ferramentas de produtividade

Alguns exemplos populares:

- Python
- Java
- SQL
- AWS
- Azure
- Git

------

### 2️⃣ Faça o teste

O teste geralmente possui:

- cerca de **15 perguntas**
- múltipla escolha
- tempo limitado

Ele avalia conhecimento **prático e conceitual**.

------

### 3️⃣ Receba seu badge

Se você atingir uma pontuação entre os **30% melhores**, o LinkedIn concede um **selo de habilidade**.

Esse selo aparece diretamente no seu perfil.

------
